# Пошаговая инструкция по установке Cookie Robot на Windows

## Требования
- Windows 10/11
- Python 3.8 или выше
- Octo Browser (установленный и запущенный)
- Интернет-соединение для загрузки зависимостей

---

## Шаг 1: Установка Python

Если Python ещё не установлен:

1. Перейдите на https://www.python.org/downloads/
2. Скачайте последнюю версию Python (например, Python 3.11 или 3.12)
3. Запустите установщик
4. **ВАЖНО:** Поставьте галочку ✅ "Add Python to PATH"
5. Нажмите "Install Now"

**Проверка установки:**
Откройте командную строку (Win + R → cmd → Enter) и введите:
```
python --version
```
Должно показать версию Python (например, Python 3.11.5)

---

## Шаг 2: Скачивание проекта

### Вариант A: Скачать через Emergent
1. В интерфейсе Emergent нажмите кнопку "Download Code" (скачать код)
2. Распакуйте архив в удобную папку, например: `C:\CookieRobot`

### Вариант B: Создать папку вручную
1. Создайте папку `C:\CookieRobot`
2. Скопируйте туда все файлы из `/app/cookie_robot/`

---

## Шаг 3: Открытие командной строки в папке проекта

1. Откройте проводник Windows
2. Перейдите в папку с проектом (например, `C:\CookieRobot\cookie_robot`)
3. В адресной строке проводника введите `cmd` и нажмите Enter
   
   ИЛИ
   
   Зажмите Shift + ПКМ в папке → "Открыть окно PowerShell здесь"

---

## Шаг 4: Создание виртуального окружения (рекомендуется)

```cmd
python -m venv venv
```

Активация виртуального окружения:
```cmd
venv\Scripts\activate
```

После активации в начале строки появится `(venv)`

---

## Шаг 5: Установка зависимостей

```cmd
pip install PyQt5 playwright requests schedule pyinstaller
```

Установка браузера для Playwright:
```cmd
playwright install chromium
```

Эта команда скачает Chromium (~150 МБ)

---

## Шаг 6: Тестовый запуск приложения

Убедитесь, что всё работает перед сборкой .exe:

```cmd
python main.py
```

Должно открыться окно приложения "Cookie Robot".
Если окно открылось - всё готово к сборке!

Закройте приложение (крестик или Alt+F4).

---

## Шаг 7: Сборка .exe файла

```cmd
pyinstaller build.spec
```

Процесс займёт 1-3 минуты. 

После завершения:
- Готовый файл: `dist\CookieRobot.exe`
- Размер файла: ~50-100 МБ

---

## Шаг 8: Запуск готового приложения

1. Перейдите в папку `dist`
2. Запустите `CookieRobot.exe`
3. При первом запуске Windows может показать предупреждение SmartScreen:
   - Нажмите "Подробнее"
   - Нажмите "Выполнить в любом случае"

---

## Настройка Octo Browser API

### Получение API токена:
1. Откройте Octo Browser
2. Перейдите в Settings (Настройки)
3. Найдите раздел "API" или "Local API"
4. Скопируйте токен

### В приложении Cookie Robot:
1. Вставьте токен в поле "API Token"
2. URL оставьте по умолчанию: `http://localhost:58888`
3. Нажмите "Connect"
4. Статус должен измениться на "Connected" (зелёный)

## Решение проблем

### Ошибка "python не является внутренней командой"
- Python не добавлен в PATH
- Решение: Переустановите Python с галочкой "Add to PATH"

### Ошибка при установке PyQt5
```cmd
pip install --upgrade pip
pip install PyQt5
```

### Ошибка "No module named 'PyQt5'"
- Убедитесь, что виртуальное окружение активировано
- Переустановите: `pip install PyQt5`

### Приложение не подключается к Octo Browser
- Убедитесь, что Octo Browser запущен
- Проверьте, что Local API включен в настройках Octo
- Попробуйте URL: `http://127.0.0.1:58888`

### Антивирус блокирует .exe
- Это ложное срабатывание (PyInstaller часто вызывает такую реакцию)
- Добавьте файл в исключения антивируса

## Структура готового приложения

После сборки вам нужен только файл `CookieRobot.exe` из папки `dist`.
Его можно скопировать куда угодно и запускать.

Рядом с .exe при первом запуске создастся:
- `config.json` - настройки (сайты, профили, параметры)
- `logs/` - папка с логами
