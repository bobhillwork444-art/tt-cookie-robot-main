# TT Cookie Robot

Browser automation tool for Octo Browser profiles.

## Features
- Multi-profile support
- Human behavior simulation
- Proxy status monitoring
- Site list management

## Installation

Download the installer for your platform:
- **Windows**: `TTCookieRobot_Setup_v1.0.0.exe`
- **macOS**: `TTCookieRobot_v1.0.0.dmg`

## Requirements
- Octo Browser installed and running
- Active Octo Browser profiles with configured proxies

## Usage
1. Launch TT Cookie Robot
2. Click "Connect" to connect to Octo Browser
3. Add profile UUIDs
4. Add target sites
5. Select profiles and click "START SELECTED"
